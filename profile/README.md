# Bluscream's Vencord Plugins

A collection of Vencord plugins developed by Bluscream for enhancing Discord functionality.

## Plugins

| Plugin                       | Description                                                                                                           | Repository                                                                                        |
| ---------------------------- | --------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| **blu-startupChannel**       | Automatically opens and scrolls to a specific channel when Discord starts                                             | [blu-startupChannel](https://github.com/bluscream-vencord-plugins/blu-startupChannel)             |
| **blu-ghosted**              | A cute ghost will appear if you don't answer their DMs (Clyde messages don't count)                                   | [blu-ghosted](https://github.com/bluscream-vencord-plugins/blu-ghosted)                           |
| **blu-guildInviteSaver**     | Automatically collects and saves invite links before leaving a guild                                                  | [blu-guildInviteSaver](https://github.com/bluscream-vencord-plugins/blu-guildInviteSaver)         |
| **blu-instantScreenshare**   | Instantly screenshare when joining a voice channel with support for desktop sources, windows, and video input devices | [blu-instantScreenshare](https://github.com/bluscream-vencord-plugins/blu-instantScreenshare)     |
| **blu-pttToggle**            | Adds a push-to-talk toggle checkbox to the microphone button context menu next to Input Profile                       | [blu-pttToggle](https://github.com/bluscream-vencord-plugins/blu-pttToggle)                       |
| **blu-homeassistant-events** | Send Discord events to HomeAssistant for automation and monitoring                                                    | [blu-homeassistant-events](https://github.com/bluscream-vencord-plugins/blu-homeassistant-events) |
| **blu-mcp**                  | Provides an MCP (Model Context Protocol) server for inspecting Discord and running JavaScript code                   | [blu-mcp](https://github.com/bluscream-vencord-plugins/blu-mcp)                                   |
| **blu-vc-log**               | Logs voice channel joins and leaves to the associated text channel                                                    | [blu-vc-log](https://github.com/bluscream-vencord-plugins/blu-vc-log)                             |
| **blu-vc-waitforslot**       | Wait for a slot to become available in a full voice channel instead of manually checking back                         | [blu-vc-waitforslot](https://github.com/bluscream-vencord-plugins/blu-vc-waitforslot)             |
| **blu-keepServers**          | Keeps track of servers you've joined and allows you to rejoin them later (Web version using localStorage)             | [blu-keepServers](https://github.com/bluscream-vencord-plugins/blu-keepServers)                   |
| **blu-keepServers.desktop**  | Keeps track of servers you've joined and allows you to rejoin them later (Desktop version using external file system) | [blu-keepServers.desktop](https://github.com/bluscream-vencord-plugins/blu-keepServers.desktop)   |
| **vc-followUser**            | Adds a follow option in the user context menu to always be in the same VC as them                                     | [vc-followUser](https://github.com/bluscream-vencord-plugins/vc-followUser)                       |
| **vc-voiceChatUtilities**    | Perform multiple actions on an entire channel (move, mute, disconnect, etc.)                                          | [vc-voiceChatUtilities](https://github.com/bluscream-vencord-plugins/vc-voiceChatUtilities)       |
